My notes:

IIFE - A function that is called immediately. This is helpful for cases where scope and garbage removal is helpful to remove variables that are not needed globally. To do so you simply add a pair of parentheses at the end of the function declaration. With ES6 code put within blocks also handle scope in the same way.
Redefining functions - A function can redefine itself through its own block of code. This is done in the function itself by assigning its own variable to a new function. This can be useful for initialization purposes. Once we know the values that we have thereâ€™s no need to reevaluate them, instead we can have the function redefine itself with the new function based on the first observation.
Pure functions - A function that does no destruction to any variables or methods outside its scope. It has to have at least one parameter and return something. These help to reduce confusion and bugs.
